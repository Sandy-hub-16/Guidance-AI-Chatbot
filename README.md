Guidance Counselor AI Chatbot is a web-based student support system that combines a rule-based recommendation engine with a conversational AI to help students navigate academic, personal, and wellness challenges.

At the start of each session, students fill out a short check-in covering their mood, sleep, academic standing, workload, social situation, and financial stress. The system's recommendation engine analyzes these inputs and identifies which areas need attention — classifying each as mild, moderate, or severe. These detected conditions are then passed to an AI-powered chatbot (backed by Groq's LLM API) that converses with the student the way a real guidance counselor would — listening, validating, asking follow-up questions, and naturally weaving in concrete suggestions from the recommendation engine as the conversation flows.

Students can also raise concerns freely in chat like group conflicts, thesis stress, personal struggles and the chatbot responds with empathy and actionable advice, the same way a campus counselor would in a one-on-one session. Every message is screened by a crisis filter before it ever reaches the AI; if a student expresses language associated with self-harm, the system immediately responds with a fixed, counselor-reviewed message and real crisis hotline information instead of an AI-generated reply.

All conversations are saved and accessible in the sidebar, and past check-in data persists so the system can track how a student's conditions evolve over time.
